Detailed Description

This class provides two kind of deploy Strategy.

The first strategy is implemented by createDepoyInfo function without device assignment, it consider of OS and processor and allocation properties. well the second strategy is implemented by ceateDeployInfo with device assignement other one without device assignments. It mainly verify device capacity is suitable for component deploy dependences.
