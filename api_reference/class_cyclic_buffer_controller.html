Detailed Description

The class managers a buffer which used to store data form bus and privide data to waveform.

The buffer has a initial size. WritePtr used to write data to the buffer, and decrease relevant value of writableSize after writing, if writableSize is less than the size of data need to write to the buffer, then will write failed. ReadPtr used to read data from buffer, and increase relevant size of writableSize after reading.
